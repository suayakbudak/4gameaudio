# 4GameAudio

A modern, responsive website built to showcase game audio services, sound design, and interactive media work.

## Live Demo
https://4gameaudio.com/

## About
4GameAudio is a clean and user-focused website designed to present audio solutions for games and digital experiences. The project emphasizes performance, accessibility, and clear content structure.

## Features
- Responsive design for mobile, tablet, and desktop
- Clean and minimal user interface
- Fast and lightweight performance
- Structured sections for presenting services and content
- Built without frameworks using core web technologies

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
